Tipos de ameaças

Entender o que existe lá fora é o primeiro passo para se proteger.

De onde vêm as ameaças?

  • Externas: atacantes de fora da organização (hackers, grupos, governos).
  • Internas: funcionários, estagiários, terceirizados — com ou sem má intenção.
  • Acidentais: derramou café no notebook, deletou sem querer, usou pendrive infectado.

Principais tipos de ataque

NomeO que é
PhishingE-mails/links falsos para roubar senhas.
MalwareQualquer software malicioso (vírus, worm, trojan…).
RansomwareCriptografa seus dados e pede resgate.
DDoSMilhares de pedidos para derrubar um serviço.
MITMAtacante "no meio" da conversa, lendo/alterando dados.
Brute forceTestar muitas senhas até acertar.
SQL InjectionEnganar o banco de dados inserindo código SQL em campos.
XSSInjetar JavaScript malicioso em um site.
Zero-dayFalha desconhecida sendo explorada antes do patch.
Engenharia socialManipular pessoas (não o software).

Atores de ameaça

  • Script kiddies: iniciantes usando ferramentas prontas.
  • Cibercriminosos: querem dinheiro.
  • Hacktivistas: motivação política/ideológica.
  • APT (Advanced Persistent Threat): grupos organizados, às vezes estatais.
  • Insiders: alguém de dentro.
Moral da história: Você não precisa ser "importante" para ser atacado. A maioria dos ataques é automatizada e atinge quem estiver vulnerável.
